### Авторизация 
Логика работы:

Проверка токена в localStorage

Если токен есть - показ имени пользователя

Если нет - отображение кнопок входа/регистрации

Эндпоинты: 

POST /auth/login - Вход  
POST /auth/register - Регистрация  
GET /user - Данные пользователя
